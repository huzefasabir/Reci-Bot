# Recipe Chatbot

A web-based AI-powered chatbot that helps users generate and customize recipes based on ingredients, dish names, and servings.

## Features
- Interactive chatbot for generating and customizing recipes
- Ingredient-based recipe suggestions
- Step-by-step cooking instructions with nutritional information
- AI-generated responses using an external API
- User-friendly UI with smooth animations

## Technologies Used
- HTML, CSS, JavaScript
- FontAwesome for icons
- External AI API for recipe generation

## Installation & Setup
1. Clone the repository:

  https://github.com/huzefasabir/Reci-Bot.git

2. Navigate to the project folder:

  cd Reci-bot/reci-bot
   
3. Open `index.html` in your browser.

## Usage
- Upon loading, the chatbot presents three main features:
  - **Recipe Creator**: Enter a dish name to get a detailed recipe.
  - **Ingredient Wizard**: Input available ingredients to generate recipes.
  - **Image to Weight**: Convert images into ingredient weight estimates.
- Interact with the chatbot to get step-by-step instructions and ingredient details.
